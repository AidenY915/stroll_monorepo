<div>
  <h3>프로젝트 소개</h3>
  산책갈까는 반려동물 동반 가능 시설들을 모아서 보여주는 웹 서비스입니다.
  <h5>주요 기능</h5> 
  1. 카카오맵을 통해 입력한 위치를 기준으로 주변의 반려동물 동반 가능 시설들을 거리 순으로 정렬해서 보여준다. <br/>
  2. 시설의 주소나 이름으로 관련 시설들을 검색해준다. <br/>
